Resumo Spring Boot
	Spring Boot é um framework Java open source, desenvolvido para fornecer um modelo de desenvolvimento mais simples e direto, onde a maioria dos componentes necessários para as aplicações em geral estão pré-configurada, possibilitando assim rapida configuração e implantação. Facilitando a configuração, por exemplo, de servidores, gerenciamento de dependências, configurações de bibliotecas, métricas e health checks.
	Para proporcionar esse serviço o framework se baseia em quatro principios: oferecer uma experiencia de inicio de projeto rápida e direta; Apresentar visão opinativa e flexível sobre a configuração dos projetos Spring; Fornecimento de requisitos não funcionais pré-configurados; Não prover geração de código e zerar a necessidade de arquivos XML.
	Sendo esses também seus benefícios, já que torna o trabalho menos laboroso mas ainda sim é flexível para alterações e utilizar arquivos de mais fácil utilização.
	Para a utilização do Spring boot com a adição de novas dependencias, como JPA e Hibernate é necessário utilizar o Spring Initializr, que ele utiliza outras IDEs e permite a criação de projetos de forma facilitada. Nele é possível inserir dados do projeto e adicionar as dependencias do projeto de acordo com a linguagem. Após o processo o projeto sera gerado com toda a configuração base.  Caso necessário novas dependencias podem ser adicionadas no arquivo pom de forma manual.
	O Spring Boot tem sua prórpia IDE chamada Spring Tool Suite (STS), que é baseada na IDE Eclipse, mas também possui uma extensão para o VSCode. Ele pode ser utilizado em qualquer IDE, mas a STS é totalmente customizada para o desenvolvimento de aplicações do ecossistema spring. Sendo assim torna-se mais especializada nessa função.
